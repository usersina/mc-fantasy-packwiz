This site hosts the stable Packwiz update channel for the Fantasy Minecraft pack. Players can import the client pack, while auto-updating instances should use the Packwiz URL.

Quick Client Setup

The public client pack is a Prism/Freesm .mrpack published as a GitHub Release artifact.

Use this import URL:

https://github.com/usersina/mc-fantasy-packwiz/releases/download/client-stable/mc-fantasy-stable.mrpack
  1. Install Prism Launcher or Freesm Launcher.
  2. Make sure the launcher uses Java 21.
  3. Choose Add Instance, then Import.
  4. Paste the import URL, or select the downloaded file.
  5. Launch the imported instance and wait for downloads to finish.
  6. Join the multiplayer server with the address you were given.

A direct .mrpack import is a snapshot. Restarting that same instance does not check for new releases. If the server pack changes later, open the instance's Modrinth tab and use Update Pack with the same URL. Re-import the URL only if that update path is unavailable.

Controls can be changed locally in Minecraft. Latest release page: GitHub Releases.

For troubleshooting and maintainer details, read the full client guide.

For Launchers

The stable Packwiz endpoint is:

https://usersina.github.io/mc-fantasy-packwiz/stable/pack.toml

Prism or Freesm updater instances should run this pre-launch command: 

"$INST_JAVA" -jar "$INST_MC_DIR/packwiz-installer-bootstrap.jar" -g -s client "https://usersina.github.io/mc-fantasy-packwiz/stable/pack.toml"

What This Site Is

  • A GitHub Pages host for the stable Packwiz channel.
  • A machine-readable update source for Packwiz Installer.
  • Not a full launcher, not a server status page, and not the Minecraft server itself.
