Fantasy Pack Setup

Import the pack once in Prism or Freesm, drop in the updater jar, add the updater command, then launch the same instance whenever you want to play.

Import .mrpack Full guide

Four Steps

1

Import The Pack

In Prism or Freesm, choose Add Instance, then Import. Paste this URL: 

https://github.com/usersina/mc-fantasy-packwiz/releases/download/client-stable/mc-fantasy-stable.mrpack

Confirm the import and let the launcher download everything.

2

Add The Updater Jar

Download packwiz-installer-bootstrap.jar and put it directly inside the imported instance's minecraft folder.

In Prism or Freesm, open the instance, then use the folder button to open its Minecraft folder.

Download updater jar
3

Turn On Updates

In the instance settings, open Custom commands and paste this as the pre-launch command: 

"$INST_JAVA" -jar "$INST_MC_DIR/packwiz-installer-bootstrap.jar" -g -s client "https://usersina.github.io/mc-fantasy-packwiz/stable/pack.toml"

This makes the instance check for pack updates before Minecraft starts.

4

Launch And Play

Launch this same instance every time. If the server owner updates the pack, close Minecraft and launch the instance again.
