# Local Mods

This folder is only for tiny compatibility patches that make this pack work.

Keep the bar high:

- Prefer upstream mods, configs, datapacks, or Packwiz metadata first.
- Keep each local mod focused on one compatibility issue.
- Do not add gameplay systems, content, blocks, items, GUIs, or broad rewrites here.
- Keep source readable enough that the jar can be audited quickly.
- Commit the source here and the built jar under `mods/`.
- If a patch grows beyond a small file or two, move it to a real standalone project.

The Packwiz client/server site ignores this folder. Only built jars in `mods/` are shipped.
